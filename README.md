## Описание

`Training Planner` — это простое десктопное приложение для планирования и отслеживания тренировок. Оно позволяет пользователям добавлять записи о своих тренировках, указывая дату, тип и длительность, а затем просматривать и фильтровать их.

### Функциональные возможности:

*   **Добавление тренировок:** Форма для ввода даты, типа и длительности тренировки в минутах.
*   **Список тренировок:** Все записи отображаются в виде таблицы, отсортированной по дате (от новых к старым).
*   **Фильтрация:** Данные можно отфильтровать по диапазону дат и по выбранному типу тренировки.
*   **Хранение данных:** Вся информация сохраняется в локальный файл `trainings.json` в формате JSON и загружается при каждом запуске.
*   **Валидация ввода:** Приложение проверяет, что длительность является положительным числом, а дата введена в корректном формате.

## Технологии

*   **Язык:** Python 3
*   **GUI:** `tkinter` и `tkcalendar`
*   **Формат данных:** JSON
*   **Контроль версий:** Git

## Как использовать

1.  **Клонирование репозитория:**
    Склонируйте или скачайте все файлы в одну папку на вашем компьютере.

2.  **Установка зависимостей:**
    Для работы приложения требуется библиотека `tkcalendar`. Установите ее с помощью `pip`:
    ```bash
    pip install tkcalendar
    ```

3.  **Запуск приложения:**
    Откройте терминал в папке с проектом и выполните команду:
    ```bash
    python training_planner.py
    ```

## Примеры использования

*   **Добавление новой тренировки:**
    1.  В форме "Добавить тренировку" выберите дату.
    2.  Выберите "Тип тренировки" из списка (например, "Кардио").
    3.  Введите "Длительность" в минутах (например, `45`).
    4.  Нажмите кнопку "Добавить". Новая запись появится в таблице ниже.

*   **Фильтрация по типу тренировки:**
    1.  В разделе "Фильтрация" выберите нужный тип из списка "Тип тренировки".
    2.  Нажмите "Применить фильтр". В таблице останутся только записи, соответствующие вашему выбору.

*   **Фильтрация по дате:**
    1.  В полях "Фильтр по дате" выберите начальную и/или конечную дату периода.
    2.  Нажмите "Применить фильтр". Таблица покажет тренировки только за указанный промежуток времени.

*   **Сброс фильтров:**
    Чтобы снова увидеть все тренировки, выберите "Все" в фильтре по типу и очистите поля с датами, после чего нажмите "Применить фильтр".
